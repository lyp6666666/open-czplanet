## ADDED Requirements

### Requirement: 首页搜索按登录态与角色路由跳转
系统 MUST 在首页搜索触发时，根据登录态与角色执行确定的跳转策略。

#### Scenario: 学生登录后使用首页搜索跳转到教师列表页
- **WHEN** 用户已登录且角色为学生，并在首页触发搜索且关键词非空
- **THEN** 系统 MUST 跳转到学生端教师列表页，并通过路由参数携带该关键词用于筛选

#### Scenario: 教师登录后使用首页搜索跳转到需求页
- **WHEN** 用户已登录且角色为教师，并在首页触发搜索且关键词非空
- **THEN** 系统 MUST 跳转到教师端需求列表页，并通过路由参数携带该关键词用于筛选

#### Scenario: 未登录使用首页搜索弹出提示
- **WHEN** 用户未登录并在首页触发搜索
- **THEN** 系统 MUST 弹出提示“请登录之后再使用搜索功能”，且 MUST NOT 跳转页面

### Requirement: 首页搜索关键词为空时不触发跳转
系统 MUST 在关键词为空或仅包含空白字符时避免触发跳转与查询。

#### Scenario: 已登录但关键词为空不跳转
- **WHEN** 用户已登录并触发首页搜索且关键词为空或仅为空白字符
- **THEN** 系统 MUST NOT 跳转页面

#### Scenario: 未登录且关键词为空不跳转
- **WHEN** 用户未登录并触发首页搜索且关键词为空或仅为空白字符
- **THEN** 系统 MUST NOT 跳转页面
